## Front-end

### 1. HTML
- HTML은 Hyper Text Markup Language의 약자.
- HTML은 웹 페이지를 만들기 위한 표준 markup language.
- HTML은 웹 페이지의 *구조를* 나타낸다.
- HTML은 element의 집합니다.
- HTML의 element는 브라우저에게 콘텐츠를 어떻게 보여줄 건인지를 말해준다.
